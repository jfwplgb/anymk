Docker

anymk 简介

WebSocket 应用场景非常广泛，例如社交聊天、弹幕、多玩家游戏、协同编辑、股票基金实时报价、体育实况更新、视频会议/聊天、实时定位、在线教育、智能家居等，这些场景都与我们的生活息息相关。

anymk 使用 Spring Framework 中的 WebSocket 模块，基于 STOMP 协议实现点对点、点对多、多对多通信等功能，并应用容器化技术加强容器编排的能力。
